## What is Copilot CLI Agent Arsenal?

A comprehensive collection of 20 pre-configured AI agents for GitHub Copilot CLI that automate your development tasks.

## Installation

```bash
npm install -g @github/copilot-cli
git clone https://github.com/RLuf/copilot-cli-arsenal.git
cd copilot-cli-arsenal
copilot-cli config agents load ./.copilot/agents
```

## 20 Specialized Agents

- **Frontend Developer** - UI/UX implementation
- **Backend Architect** - API & database design
- **Code Reviewer** - Quality assurance
- **DevOps Engineer** - Infrastructure automation
- **Security Specialist** - Vulnerability analysis
- ...and 15 more specialized agents

## Quick Start

```bash
copilot-cli agents enable --all
copilot-cli start
```
